## Installation & Set Up
Clone down this repository:

`$ git clone https://github.com/jaykmark/homevision.git`

You will need `node` and `npm` or some package manager installed on your machine.  

Installation (w/ npm):

`$ npm install`  

To Start Server (w/ npm):

`$ npm start`  

To Visit App:

`localhost:3000`
